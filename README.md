## Features

### Backend
1. **CRUD Operations**:
   - Add, view, update, and delete books.
2. **Search Functionality**:
   - Search books by title, author, or genre.
3. **Detailed Views**:
   - Display detailed information about a specific book.
4. **Validation**:
   - Validate API inputs for data integrity.

### Frontend
1. **User-Friendly Interface**:
   - Display book lists and details.
2. **Search and Filter**:
   - Search books by various criteria.
3. **Add/Edit/Delete**:
   - Librarians can manage book collections.
